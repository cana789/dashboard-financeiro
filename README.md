# Dashboard de Execução Financeira v2 — Design BI
## 4º Btl Op Lit FN — Exercício 2026

Versão redesenhada: tema dark BI com KPIs em destaque, gauges e cores neon.

## Arquivos
- `app.py` — aplicação completa
- `requirements.txt` — dependências fixas (streamlit 1.45.1, pandas 2.2.3, plotly 5.24.1)
- `.python-version` — fixa Python 3.11
- `.streamlit/config.toml` — tema dark nativo

## Deploy no Streamlit Cloud
Suba TODOS os arquivos para o GitHub mantendo a estrutura:
```
dashboard-financeiro/
├── app.py
├── requirements.txt
├── .python-version
└── .streamlit/
    └── config.toml
```
A pasta `.streamlit` com o `config.toml` garante o tema dark correto.
